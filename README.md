
# Testes

Hi, in this repository I created my project using the api of one of my favorite Elden ring games, I'll leave the link to the api I used along with the technologies I used and how to install🚀🚀.


## 🛠️ Built with



    React

    javaScript

    Jest

    Husky

    Styleed-components

Package manager

    yarn



## Installation

Type in your terminal to start installing the dependencies

    yarn 

## Run locally

Tests in Jest

    yarn test

Jest tests with coverage

    yarn coverage

Tests in cypress

    yarn cypress
